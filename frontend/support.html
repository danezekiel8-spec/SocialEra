What you need first

Google requires a Cloud project, Gmail API access, and a web OAuth client before a website can connect to your inbox.

1. Enable Gmail API Turn on Gmail API inside a Google Cloud project you control.
2. Create a web client ID Add your local and future production site URLs to the authorized origins list.
3. Paste your keys here This page stores them only in your browser so you can test the support flow locally first.

Gmail Setup
